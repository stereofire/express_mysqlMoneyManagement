# express_mysqlMoneyManagement
nodejs实践的高校学生资金综合管理平台学生端

安装依赖： npm i
运行：npm start
结束批处理进程： Ctrl + c

### /bin: 用于应用启动
### /public: 静态资源目录
### /routes：可以认为是controller（控制器）目录
### /views: ejs模板目录，可以认为是view(视图)目录

### /util – 工具方法
### /conf – 配置
### /dao – 与数据库交互
### /node_modules 依赖
### app.js 程序main文件
### package.json 依赖记录文件
